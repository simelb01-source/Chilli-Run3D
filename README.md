# 🌶️ Chili-Run 3D 🔥

Ein flottes, endloses 3D-Runner-Spiel, das komplett im Browser läuft. Steuere eine feurige Chili durch eine gefährliche Küche, weiche Töpfen, Messern und Flammen aus und sammle Punkte, während das Spiel immer schneller wird!

Dieses Projekt wurde mit **Three.js** und reinem Vanilla-JavaScript (ohne Build-Tools) umgesetzt.

## 🎮 Gameplay & Features

*   **Endloser Runner:** Laufe so lange du kannst, während das Tempo stetig steigt.
*   **Dynamische Hindernisse:** Weiche drei verschiedenen Hindernistypen aus (Töpfe, Messer, flackernde Flammen).
*   **3-Lane-System:** Weiche flüssig auf drei Spuren nach links und rechts aus.
*   **Springen:** Überwinde Hindernisse mit einem gut getimten Sprung.
*   **Mobile-First:** Vollständige Touch- & Swipe-Steuerung für Smartphones und Tablets.
*   **Schatten & Licht:** Echtes Beleuchtungssetup mit dynamischen Schatten für mehr Tiefe.
*   **Faire Kollision:** Präzise `Box3`-Kollisionserkennung für ein faires Spielgefühl.
*   **Performance-Optimiert:** Geometrien und Materialien werden gecacht, um eine konstante Framerate zu gewährleisten.

---

## 🕹️ Steuerung

### Desktop (Tastatur)
*   **Links / Rechts** oder **A / D**: Spur wechseln
*   **Leertaste**, **Pfeil nach oben** oder **W**: Springen

### Mobile (Touch)
*   **Nach links / rechts wischen**: Spur wechseln
*   **Nach oben wischen**: Springen

## 🚀 Installation & Lokales Spielen

Da dieses Projekt keine Build-Tools (wie Webpack oder Vite) verwendet, kannst du es ganz einfach lokal ausführen:

1. Klone das Repository:
   ```bash
   git clone https://github.com/DEIN_USERNAME/chili-run-3d.git
   ```
2. Navigiere in den Projektordner:
   ```bash
   cd chili-run-3d
   ```
3. Öffne die `index.html` Datei einfach per Doppelklick in deinem Lieblingsbrowser.

*Alternativ:* Wenn du Live-Reload möchtest, nutze die VS Code Erweiterung "Live Server" und klicke auf "Go Live".
